The Web Programming course introduces us to how to create and design a website. It covers front-end technologies like HTML, CSS, and JavaScript.

  1. At the beginning lecture was about
    • The Internet and Web Technologies:
      • Basics of the Internet and the World Wide Web (WWW)
      • Client-server architecture and HTTP protocols
  2. Front-End Development:
    • Designing and developing web pages using HTML, CSS, and JavaScript
  3. Back-End Development:
    • Introduction to server-side programming using PHP
  4. Web Design Principles:
    • Modern web design practices (Web 2.0, Web 3.0)
  5. Web Project Lifecycle:
    • From conceptual design to implementation
    • Testing, publishing, and maintaining websites
